# Jack Kweyunga's Professional Portfolio

A modern, minimalist portfolio showcasing my work as a DevSecOps Engineer, Electronics Engineer, and Co-founder of Seede XR.

## Tech Stack

- Next.js 15 with TypeScript
- Tailwind CSS with custom themes
- MDX for content editing
- next-themes for dark/light mode

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Features

- Responsive design with dark/light mode toggle
- MDX support for easy content editing
- Professional emerald green color scheme
- Optimized for performance and accessibility

## Deployment

Deploy easily on Vercel, Netlify, or any static hosting platform.
